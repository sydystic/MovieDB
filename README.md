🎬 MovieDB – IMDb Clone

A responsive, modern movie database web app built with **React** and styled using **Tailwind CSS**, powered by the **TMDb API**. MovieDB allows users to search, explore, and browse details of thousands of films in an elegant interface.

[🌐 Live Site](https://sidsmoviedb.netlify.app)


## 🛠️ Tech Stack

- **Frontend**: React.js
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Data Source**: TMDb API
- **Deployment**: Netlify


## 📸 Features

- 🔍 **Search for Movies** by title
- 🎞️ **View Details** including poster, release year, rating, and synopsis
- 📃 **Pagination** for browsing multiple pages of results
- 🎨 **Responsive Design** for mobile and desktop
- ⚡ **Fast Loading** with optimized API calls
- ✨ **Smooth UI Animations** using Framer Motion
