MovieDB is a simple movie database web application built to search and explore films in a clean interface. It uses React for the frontend and Tailwind CSS for styling, with data powered by the TMDb API. The app allows users to search for movies by title and view basic details such as posters, release year, ratings, and summaries. Results are displayed across multiple pages, making it easier to browse larger sets of data. The interface is responsive and works across different screen sizes, with small animations added to keep interactions smooth without overcomplicating the experience. Some features are not fully polished and a few edge cases may not behave perfectly, but the project focuses on building a functional UI, working with external APIs, and understanding how to structure and deploy a real-world frontend application.
